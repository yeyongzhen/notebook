## 一、PHP基础
### 1、注释
PHP支持三种注释方式：
- 单行 C++注释
- shell语法注释
- 多行C语法注释

```
<?php
// 单行注释
echo ”This is a PHP program";

# shell语法注释
echo "This is shell annotation";

/*
  这是多行注释
*/
echo "This is duplicate annotation";

```

### 2、支持的数据类型
PHP 支持 9 种原始数据类型。

四种**标量**类型：
- boolean（布尔型）
- integer（整型）
- float（浮点型）
- string（字符串）

三种**复合**类型：
- array（数组）
- object（对象）
- callable（可调用）

两种**特殊**类型：
- resource（资源）
- NULL（无类型）
